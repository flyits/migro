Migro

Go 语言数据库迁移工具，提供 Laravel 风格的优雅 API，让数据库迁移变得简单。

快速入门 GitHub

特性

🔗 流畅的链式 API

Laravel 风格的 Schema DSL，代码简洁易读，让数据库表定义变得优雅。

🗄️ 多数据库支持

支持 MySQL、PostgreSQL、SQLite，一套代码适配多种数据库。

🔒 事务保护

PostgreSQL/SQLite 支持事务 DDL，确保迁移的原子性和数据安全。

⚙️ 环境变量支持

配置文件支持 ${VAR:default} 语法，轻松管理多环境配置。

📦 批次管理

支持按批次回滚迁移，精确控制数据库版本。

👁️ Dry Run 模式

预览 SQL 而不执行，安全地检查迁移内容。

🔌 外部连接支持

支持传入已有的 *sql.DB 或 GORM 连接，轻松集成现有项目。

安装

使用 Go 安装：

bash 
go install github.com/flyits/migro/cmd/migro@latest

或者从源码构建：

bash 
git clone https://github.com/flyits/migro.git
cd migro
go build -o migro ./cmd/migro

快速示例

创建一个用户表的迁移：

go 
func (m *CreateUsersTable) Up(ctx context.Context, e *migrator.Executor) error {
    return e.CreateTable(ctx, "users", func(t *schema.Table) {
        t.ID()
        t.String("name", 100)
        t.String("email", 100).Unique()
        t.String("password", 255)
        t.Timestamps()
    })
}
